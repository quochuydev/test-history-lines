## About

This tool analyzes cryptocurrency candlestick data to find the optimal single buy/sell transaction that would yield maximum profit. The analysis uses a linear-time algorithm to identify the best entry and exit points.

### Tracked Assets
- **BTC/USDT** - Bitcoin
- **ETH/USDT** - Ethereum
- **SOL/USDT** - Solana

### Analysis Intervals
- 15-minute candles
- 1-hour candles
- Daily candles
